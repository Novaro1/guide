Sign in with a GitHub Personal Access Token

How to set this up (one time, ~2 min)
  1. Open GitHub → fine-grained tokens.
  2. Set Resource owner = your account, Repository access = "Only select repositories" → pick this guide's repo.
  3. Under Repository permissions, set Contents = Read and write. Leave everything else.
  4. Set an expiration (90 days recommended). Generate, copy the token (starts with github_pat_).
  5. Paste it below along with your owner/repo.

Why this is secure: only you can generate this token, it's scoped to one repo and one permission, you can revoke it instantly from GitHub, and nothing is stored on a server — it lives in your browser only.
